# The Hidden Cost of Accuracy: An Energy-Aware Evaluation of Machine Learning Models for Weather Prediction

Does higher machine learning accuracy justify the environmental cost? This project evaluates the trade-offs between model complexity, accuracy, and carbon footprints in the context of weather prediction. We train and compare a computationally lightweight **Linear Regression** model against a more complex **Random Forest Regression** model to determine if the marginal gains in accuracy are worth the extra energy consumption.

## Tech Stack
- **Language:** Python
- **Data Manipulation:** Pandas
- **Machine Learning:** Scikit-learn
- **Carbon Tracking:** CodeCarbon

## Getting Started

### Prerequisites
Make sure you have Python installed, then install the required dependencies:
```bash
pip install pandas scikit-learn codecarbon
```

### Installation & Execution
1. Clone this repository or download the source files.
2. Open your preferred code editor or terminal in the project directory.
3. Run the script using the following command:
```bash
python main.py
```
or replace main.py with your own Python file
